# 使用说明

## 1. 登录

访问系统后自动跳转到登录页面。

- **默认管理员账号：** admin / admin123
- 输入用户名和密码后点击「登录」或按 Enter 键
- 登录成功后跳转到首页

---

## 2. 首页

首页展示系统概览信息：

- **统计卡片：** 菜单数量、页面配置数、字段配置总数
- **快捷入口：** 菜单管理、页面配置的快速跳转链接
- **系统信息：** 技术栈和主要功能介绍

---

## 3. 数据页面

通过左侧菜单导航到各业务数据页面。每个数据页面根据页面配置自动生成表单和表格。

### 3.1 查看数据

- 页面加载后自动展示该集合的所有数据
- 表格列根据字段配置自动生成
- 序号列自动编号
- reference 字段的继承字段会在表格中额外显示

### 3.2 新增数据

1. 点击页面右上角「+ 新增」按钮
2. 在弹出的对话框中填写表单
3. 必填字段会有红色星号标识
4. 点击「确定」保存，「取消」放弃

### 3.3 编辑数据

1. 在表格操作列点击「编辑」
2. 修改表单中的字段值
3. 点击「确定」保存修改

### 3.4 删除数据

1. 在表格操作列点击「删除」
2. 确认删除对话框中点击「删除」
3. 如果该记录被其他数据引用（reference），会提示无法删除

### 3.5 导出 Excel

点击「导出」按钮，浏览器自动下载 .xlsx 文件。导出内容：
- 所有字段按配置顺序排列
- 下拉选项类字段显示为中文标签（而非值）
- 多选字段用 「、」分隔

### 3.6 导入 Excel

1. 点击「导入」下拉按钮
2. 选择「下载导入模板」获取 Excel 模板文件
3. 模板包含两个工作表：
   - **导入数据：** 按表头填写数据
   - **字段说明：** 字段名、类型、是否必填、可选值说明
4. 填写完成后，点击「导入」→「导入数据」选择文件
5. 系统解析文件并逐条导入，显示进度和结果

---

## 4. 菜单管理

> 仅管理员可访问（系统配置 → 菜单管理）

管理侧边栏的菜单结构，支持最多 3 级嵌套。

### 4.1 页面布局

- **左侧：** 菜单树，展示当前菜单层级结构
- **右侧：** 选中菜单的编辑表单

### 4.2 新增菜单

- 点击「新增顶级菜单」创建根菜单
- 鼠标悬停在菜单节点上，点击「添加子菜单」创建子菜单
- 填写菜单信息后点击「创建」

### 4.3 编辑菜单

点击菜单树中的节点，右侧显示编辑表单：

| 字段 | 说明 |
|------|------|
| 菜单名称 | 侧边栏显示的名称（2-20 字符） |
| 菜单图标 | 从 Element Plus 图标库选择 |
| 路由路径 | 如 /inspection/case |
| 关联页面 | 叶子菜单需选择对应的页面配置 |
| 排序序号 | 控制同级菜单的显示顺序 |
| 可见角色 | 勾选 admin / developer / guest |

### 4.4 删除菜单

鼠标悬停在菜单节点上，点击「删除」，确认后删除。有子菜单时需先删除子菜单。

---

## 5. 页面配置

> 仅管理员可访问（系统配置 → 页面配置）

定义数据页面的字段结构。每个页面配置对应一个数据集合。

### 5.1 页面布局

- **左侧：** 页面列表，支持搜索
- **右侧：** 页面基本信息 + 字段配置编辑器

### 5.2 创建页面

1. 点击左侧列表头部的「新增」按钮
2. 填写页面名称、描述、API 端点
3. 点击「创建」后自动选中新页面
4. 在字段编辑器中添加字段

### 5.3 字段配置

点击「添加字段」打开字段编辑对话框：

| 配置项 | 说明 |
|--------|------|
| 显示名称 | 表单标签和表格列标题 |
| 字段名 | 存储字段名（英文，如 caseName） |
| 控件类型 | 见下方控件类型列表 |
| 必填 | 表单提交时是否必须填写 |
| 主键 | 标记为唯一标识字段（支持复合主键） |
| 占位提示 | 输入框的提示文字 |

### 5.4 支持的控件类型

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| text | 单行文本 | 名称、编号 |
| textarea | 多行文本 | 描述、备注 |
| number | 数值 | 数量、金额 |
| select | 单选下拉 | 状态、类型 |
| multiSelect | 多选下拉 | 标签、多项选择 |
| radio | 单选按钮 | 二选一场景 |
| checkbox | 复选框 | 多项勾选 |
| date | 日期 | 日期字段 |
| datetime | 日期时间 | 精确到时分秒的时间字段 |
| file | 文件上传 | 文档附件（最多 5 个，单个 10MB） |
| image | 图片上传 | 照片（最多 9 张，单张 5MB） |
| relation | 多对多关联 | 与其他数据集合建立双向关联 |
| reference | 一对多引用 | 引用父记录，支持继承字段显示 |

### 5.5 选项配置（select / multiSelect / radio / checkbox）

支持两种选项来源：
- **静态选项：** 手动添加 标签-值 对
- **API 获取：** 配置 API 地址和标签/值字段名

### 5.6 关联字段配置（relation）

| 配置项 | 说明 |
|--------|------|
| 目标集合 | 要关联的数据集合 |
| 显示字段 | 下拉列表中显示的字段 |
| 反向字段 | 在目标集合中自动维护的反向关联字段名 |

关联是双向的：修改 A→B 的关联会自动同步 B→A。

### 5.7 引用字段配置（reference）

| 配置项 | 说明 |
|--------|------|
| 目标集合 | 引用的父数据集合 |
| 显示字段 | 下拉列表中显示的字段 |
| 继承字段 | 在数据表格中额外显示的父记录字段 |

引用会建立依赖：被引用的父记录无法删除。

---

## 6. 用户管理

> 仅管理员可访问（系统配置 → 用户管理）

### 6.1 用户列表

表格展示所有用户的用户名、显示名称、角色、创建时间。

### 6.2 新增用户

1. 点击「新增用户」
2. 填写用户名、显示名称、角色、密码（≥6 位）
3. 点击「确定」创建

### 6.3 编辑用户

点击「编辑」修改显示名称和角色（用户名不可修改）。

### 6.4 重置密码

点击「重置密码」，输入新密码（≥6 位）后确认。

### 6.5 删除用户

点击「删除」并确认。当前登录用户不可删除自己。

### 6.6 角色说明

| 角色 | 权限范围 |
|------|---------|
| admin | 所有功能，包括系统配置 |
| developer | 业务数据的增删改查、导入导出 |
| guest | 业务数据的查看（菜单可见性由管理员配置） |

---

## 7. 操作日志

> 仅管理员可访问（系统配置 → 操作日志）

记录系统中所有新增、修改、删除操作的审计跟踪。

### 7.1 查看日志

表格展示：操作描述、操作类型、目标类型、操作人、角色、操作时间。

### 7.2 筛选

支持以下筛选条件（组合使用）：

| 筛选项 | 说明 |
|--------|------|
| 时间范围 | 选择开始和结束时间 |
| 操作类型 | 新增 / 修改 / 删除 |
| 目标类型 | 菜单 / 页面配置 / 动态数据 / 用户 / 关联关系 |
| 操作人 | 输入用户名搜索 |

点击「搜索」应用筛选，「重置」清除所有条件。

### 7.3 导出

点击「导出」按钮，将当前筛选结果导出为 Excel 文件。

### 7.4 删除日志

点击操作列的「删除」可删除单条日志记录（确认后执行）。

### 7.5 分页

支持每页 20 / 50 / 100 条，底部显示总数和翻页控件。

---

## 8. 系统备份

> 仅管理员可访问（系统配置 → 系统备份）

### 8.1 手动备份

1. 点击「创建备份」按钮
2. 可选填写备注信息（如「部署前备份」）
3. 点击「确认创建」
4. 备份完成后出现在列表中，显示文件大小和记录数

### 8.2 下载备份

点击备份列表中的「下载」按钮，浏览器下载 .zip 文件。ZIP 文件内包含：
- `manifest.json` — 备份元信息
- 6 个 JSON 文件 — 各表数据

### 8.3 从备份还原

1. 点击备份列表中的「还原」按钮
2. 系统弹出警告：**还原将覆盖当前所有业务数据**
3. 确认后执行还原
4. 还原成功后页面自动刷新

> 还原是原子操作：要么全部成功，要么全部回滚，不会出现数据不一致。

### 8.4 上传还原

1. 点击「上传还原」按钮
2. 选择之前下载的 .zip 备份文件
3. 确认警告后执行还原
4. 成功后页面自动刷新

适用于跨环境迁移数据（如从测试环境恢复到生产环境）。

### 8.5 定时备份

在页面顶部的定时备份设置卡片中配置：

| 设置 | 说明 |
|------|------|
| 启用定时备份 | 开关控制是否自动执行 |
| 备份周期 | 每天 / 每周 / 每月 |
| 保留数量 | 自动清理超出数量的旧备份（默认 10） |

修改后点击「保存设置」生效。启用后系统后台自动在间隔到期时创建备份。

### 8.6 删除备份

点击「删除」按钮并确认，备份文件和记录同时删除。

## 9. 常见操作流程

### 流程一：创建一个新的业务数据页面

1. **页面配置** → 点击「新增」→ 输入页面名称和 API 端点 → 创建
2. 在字段编辑器中逐个添加字段 → 配置字段类型和属性 → 保存配置
3. **菜单管理** → 选择父菜单 → 添加子菜单 → 填写名称、图标、路由路径 → 关联刚创建的页面 → 设置可见角色 → 创建
4. 刷新页面后侧边栏出现新菜单，点击即可进入数据页面

### 流程二：建立两个数据集合的关联

1. **页面配置** → 选择源页面 → 添加 relation 类型字段
2. 配置目标集合、显示字段、反向字段名 → 保存
3. 进入源数据页面 → 编辑记录 → 在关联字段下拉框中选择目标记录
4. 关联自动双向同步

### 流程三：建立父子引用关系

1. **页面配置** → 选择子页面 → 添加 reference 类型字段
2. 配置目标集合（父集合）、显示字段、继承字段 → 保存
3. 进入子数据页面 → 新增记录 → 在引用字段中选择父记录
4. 父记录的继承字段自动在表格中显示
5. 被引用的父记录受到删除保护

### 流程四：批量导入数据

1. 进入数据页面 → 点击「导入」→「下载导入模板」
2. 在 Excel 中按模板格式填写数据（参考「字段说明」工作表）
3. 点击「导入」→「导入数据」→ 选择文件
4. 查看导入结果（成功/失败条数）

### 流程五：备份和迁移数据

1. **系统备份** → 创建手动备份 → 下载 ZIP 文件
2. 在目标环境部署系统 → 初始化数据库
3. **系统备份** → 上传还原 → 选择下载的 ZIP 文件
4. 确认还原 → 数据完整迁移

---

## 10. 注意事项

1. **密码安全：** 首次部署后请立即修改默认管理员密码
2. **备份策略：** 建议启用每日定时备份，保留至少 7 天
3. **删除操作：** 删除菜单前需先删除子菜单；删除被引用的数据会被阻止
4. **字段名规范：** 字段名只能使用英文字母、数字和下划线，以字母开头
5. **角色分配：** 遵循最小权限原则，非必要不授予 admin 角色
6. **浏览器支持：** 推荐使用 Chrome、Edge 等现代浏览器
