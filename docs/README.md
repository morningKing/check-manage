# Check-Manage 文档中心

本目录包含Check-Manage系统的完整技术文档，按类型分类组织。

## 目录结构

### 📘 [user-guide/](./user-guide/) - 用户使用手册
面向最终用户的功能使用文档，包含：
- 核心功能使用指南
- 查询与分析功能说明
- 系统集成与扩展指南
- 部署与运维手册

### 📕 [design/](./design/) - 设计文档
面向开发人员的技术设计文档，包含：
- 系统架构设计
- 功能模块设计文档
- 技术优势分析报告

### 📗 [issues/](./issues/) - 问题修复与优化
历史问题修复记录和系统优化文档，包含：
- 版本管理相关问题
- Bug修复验证报告
- 性能优化方案

### 📙 [superpowers/](./superpowers/) - 开发流程文档
使用Superpowers开发流程生成的规范文档，包含：
- `plans/` - 实现计划文档
- `specs/` - 设计规范文档
