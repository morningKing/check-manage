# 使用说明

## 1. 登录

访问系统后自动跳转到登录页面。

- **默认管理员账号：** admin / admin123
- 输入用户名和密码后点击「登录」或按 Enter 键
- 登录成功后跳转到首页

---

## 2. 首页

首页展示系统概览信息：

- **统计卡片：** 菜单数量、页面配置数、字段配置总数
- **快捷入口：** 菜单管理、页面配置的快速跳转链接
- **系统信息：** 技术栈和主要功能介绍

---

## 3. 数据页面

通过左侧菜单导航到各业务数据页面。每个数据页面根据页面配置自动生成表单和表格。

### 3.1 视图切换

数据页面支持多种视图模式：

| 视图 | 说明 |
|------|------|
| **表格视图** | 传统表格，支持排序、筛选、分页 |
| **Excel 视图** | Univer 电子表格，支持冻结、筛选、导出 |
| **看板视图** | 拖拽式卡片布局，需在页面配置中启用 |

点击工具栏右侧的视图切换按钮即可切换。

### 3.2 表格视图

#### 查看数据

- 页面加载后自动展示该集合的所有数据，默认每页 50 条
- 支持关键字搜索（模糊匹配文本、数值、选项、日期等字段）
- 点击表头可排序
- reference 字段的继承字段会在表格中额外显示
- autoTimestamp 和 autoSequence 字段只读展示

#### 查看记录详情

1. 在表格操作列点击「查看」按钮
2. 弹出的对话框展示该记录的所有字段值
3. 各字段类型的展示方式：
   - **文本/数值/序列：** 直接显示值
   - **多行文本：** 保留换行格式
   - **选项类：** 显示选项标签
   - **多选类：** 以 Tag 标签展示
   - **关联关系：** 以 Tag 展示，点击可跳转
   - **引用选择：** 以 Tag 展示，点击可跳转
   - **数据引用：** 显示父记录名称，点击可跳转
   - **文件：** 显示文件名列表，可下载
   - **图片：** 缩略图展示，支持预览

### 3.3 Excel 视图

Excel 视图基于 Univer 电子表格引擎，提供真实的表格体验：

| 功能 | 说明 |
|------|------|
| **工具栏** | 导出按钮 |
| **公式栏** | 显示单元格内容 |
| **列筛选** | 点击列头下拉菜单筛选 |
| **列宽拖拽** | 拖拽列边界调整宽度 |
| **冻结窗格** | 首列和表头自动冻结 |
| **双击导航** | 双击关联/引用字段跳转到目标记录 |

**只读模式：** Excel 视图为只读，不可编辑数据。

### 3.4 看板视图

看板视图将数据以卡片形式按列展示：

1. 在页面配置中启用看板并配置分组字段
2. 切换到看板视图
3. 拖拽卡片可在列之间移动（自动更新分组字段值）
4. 点击卡片可查看详情

### 3.5 新增数据

1. 点击页面右上角「+ 新增」按钮
2. 填写表单（必填字段有红色星号标识）
3. autoTimestamp 和 autoSequence 字段自动填充，不在表单中显示
4. 如果绑定了校验脚本，提交时自动执行
5. 点击「确定」保存

### 3.6 编辑数据

1. 点击「编辑」进入编辑模式
2. 修改字段值
3. autoTimestamp 自动更新，autoSequence 保持不变
4. 点击「确定」保存

### 3.7 删除数据

- **单条删除：** 点击「删除」，确认后执行
- **批量删除：** 选中多条记录，点击「批量删除」（仅管理员）
- 如果记录被其他数据引用，会提示无法删除

### 3.8 导出 Excel

点击「导出」按钮下载 .xlsx 文件：
- 所有字段按配置顺序排列
- 下拉选项显示中文标签
- 多选字段用「、」分隔

### 3.9 自定义脚本导出

如果页面绑定了导出脚本，「导出」按钮会变为下拉菜单：
- **导出 Excel**：默认导出
- **脚本名称**：执行自定义导出脚本

### 3.10 行级导出

如果页面绑定了行级导出脚本，每行操作列会出现「导出」按钮：
- 单个脚本：直接显示按钮
- 多个脚本：显示下拉菜单选择

### 3.11 导入 Excel

1. 点击「导入」→「下载导入模板」获取模板
2. 在 Excel 中按模板格式填写数据
3. 点击「导入」→「导入数据」选择文件
4. 查看导入结果

### 3.12 数据对比

> 仅管理员可用

1. 点击「数据对比」按钮
2. 选择基准数据和对比数据（可选当前数据或历史备份）
3. 点击「开始对比」
4. 查看新增、删除、修改的差异详情
5. 可导出对比报告

### 3.13 版本管理

> 仅管理员可用

点击「版本管理」按钮打开版本管理面板：

#### 创建快照

1. 点击「创建快照」
2. 输入快照名称（如「部署前快照」）
3. 系统保存当前数据的点-in-time 副本

#### 创建分支

1. 点击「创建分支」
2. 输入分支名称
3. 可在分支上进行数据修改，不影响主分支

#### 切换分支

点击分支名称即可切换到该分支的数据视图。

#### 合并版本

1. 选择要合并的源版本
2. 点击「合并」
3. 系统显示差异：新增、删除、修改的记录
4. 逐记录选择要合并的内容
5. 对修改记录可逐字段选择保留哪个值
6. 确认后执行合并

### 3.14 命令面板

按 `Cmd+K`（Mac）或 `Ctrl+K`（Windows）打开命令面板：

- 输入关键字搜索记录
- 快速跳转到数据页面
- 查看最近访问记录

### 5.4 编辑菜单

| 字段 | 说明 |

## 6. 页面配置

> 系统配置 → 平台管理 → 页面配置

定义数据页面的字段结构。

### 6.1 页面布局

- **左侧：** 页面列表
- **右侧：** 基本信息 + 字段编辑器

### 6.2 字段配置

| 配置项 | 说明 |
|--------|------|
| 显示名称 | 表单标签和表格列标题 |
| 字段名 | 存储字段名（英文） |
| 控件类型 | 见下方控件类型列表 |
| 必填 | 是否必须填写 |
| 主键 | 唯一标识字段 |
| 隐藏 | 在表格中隐藏 |

### 6.3 支持的控件类型

| 类型 | 说明 |
|------|------|
| text | 单行文本 |
| textarea | 多行文本 |
| richText | 富文本编辑器 |
| number | 数值输入 |
| select | 单选下拉 |
| multiSelect | 多选下拉 |
| radio | 单选按钮组 |
| checkbox | 复选框组 |
| date | 日期选择 |
| datetime | 日期时间选择 |
| file | 文件上传 |
| image | 图片上传 |
| relation | 多对多关联 |
| reference | 一对多引用 |
| quoteSelect | 引用选择 |
| autoTimestamp | 自动时间戳 |
| autoSequence | 自增序列 |

### 6.4 选项配置（select/multiSelect/radio/checkbox）

支持三种选项来源：
- **静态选项：** 手动添加标签-值对
- **API 获取：** 配置 API 地址
- **数据页数据：** 从已有数据页动态获取

### 6.5 关联字段配置（relation）

| 配置项 | 说明 |
|--------|------|
| 目标集合 | 要关联的数据集合 |
| 显示字段 | 下拉列表显示的字段 |
| 反向字段 | 目标集合中的反向关联字段名 |

关联是双向的，自动同步。

### 6.6 引用字段配置（reference）

| 配置项 | 说明 |
|--------|------|
| 目标集合 | 引用的父数据集合 |
| 显示字段 | 下拉列表显示的字段 |
| 继承字段 | 表格中额外显示的父记录字段 |

引用建立依赖：被引用的父记录无法删除。

### 6.7 自增序列配置（autoSequence）

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 前缀 | 序列号前固定字符串 | `IC-` |
| 上限 | 数字部分最大值 | `999` |

生成格式：`IC-001`、`IC-002`...

### 6.8 看板配置

在「看板配置」标签页中：
1. 开启看板功能
2. 选择分组字段（决定卡片按什么分组）
3. 选择排序字段（决定卡片在列内的顺序）
4. 配置各列的标签和颜色

### 6.9 删除绑定配置

在「删除绑定」标签页中：
1. 开启删除绑定功能
2. 选择目标集合
3. 配置删除时弹出的对话框标题和宽度
4. 配置继承字段映射

删除绑定功能：删除父记录时，可自动创建子记录或执行其他操作。

### 6.10 脚本绑定

- **校验脚本：** 新增/编辑时自动执行校验
- **导出脚本：** 页面级或行级导出脚本

### 6.11 Open API 开关

开启「允许 Open API 访问」后，外部系统可通过 API Key 访问该集合数据。

## 8. 导出脚本管理

> 系统配置 → 数据工具 → 导出脚本

### 8.1 创建脚本

| 字段 | 说明 |
|------|------|
| 脚本名称 | 显示在导出菜单中的名称 |
| 输出格式 | json / csv / xml / txt / html |
| 导出维度 | 整页数据 或 单行数据 |
| 脚本代码 | Python 代码 |

### 8.2 脚本编写

**输入变量：**

| 变量 | 类型 | 说明 |
|------|------|------|
| data | list[dict] | 业务数据列表 |
| fields | list[dict] | 字段配置列表 |
| page_name | str | 页面名称 |

**输出变量：**

| 变量 | 类型 | 说明 |
|------|------|------|
| result | str/bytes | **必须** — 导出文件内容 |
| filename | str | 可选 — 文件名 |
| content_type | str | 可选 — MIME 类型 |

**可用模块：** json, csv, io, re, math, collections, xml.etree, datetime, pandas (pd), numpy (np)

**安全限制：** 禁止 import、open()、exec()、eval()，超时 60 秒（菜单级导出 300 秒）。

## 9. 校验脚本管理

> 系统配置 → 数据工具 → 校验脚本

### 9.1 脚本编写

**输入变量：**

| 变量 | 类型 | 说明 |
|------|------|------|
| data | dict | 当前提交的记录数据 |
| fields | list[dict] | 页面字段配置 |
| all_data | list[dict] | 该集合所有现有记录 |
| is_edit | bool | 是否编辑操作 |

**校验函数：**

| 函数 | 说明 |
|------|------|
| add_error(field, message) | 添加错误，阻止提交 |
| add_warning(field, message) | 添加警告，不阻止提交 |

**示例：**

```python
# 检查名称唯一性
for existing in all_data:
    if existing.get('name') == data.get('name'):
        if is_edit and existing.get('id') == data.get('id'):
            continue
        add_error('name', '名称已存在')
        break
```

## 17. 常见操作流程

### 流程一：创建新业务数据页面

1. **页面配置** → 新增 → 定义字段 → 保存
2. **菜单管理** → 添加子菜单 → 关联页面 → 设置角色
3. 刷新页面，新菜单出现

### 流程二：建立数据关联

1. **页面配置** → 选择源页面 → 添加 relation 字段
2. 配置目标集合、显示字段、反向字段名
3. 数据页中编辑记录时选择关联记录

### 流程三：配置看板视图

1. **页面配置** → 选择页面 → 看板配置标签页

## 18. 注意事项

1. **密码安全：** 首次部署后立即修改默认管理员密码
2. **备份策略：** 建议启用每日定时备份，保留至少 7 天
3. **字段名规范：** 只能使用英文字母、数字和下划线，以字母开头
4. **角色分配：** 遵循最小权限原则
5. **浏览器支持：** 推荐 Chrome、Edge 等现代浏览器
6. **系统菜单：** 系统配置菜单为内置菜单，不可修改或删除
7. **脚本安全：** 所有脚本在沙箱中执行，超时 60 秒自动终止（菜单级导出 300 秒）
8. **API Key 安全：** 创建后仅显示一次，不用的密钥及时禁用
9. **ETL 执行：** 建议先测试运行，确认无误后再正式执行
10. **自增序列：** 删除中间记录不会回收编号
11. **并发编辑：** 系统通过乐观锁保护数据一致性
12. **版本管理：** 合并前建议先创建快照，便于回滚
