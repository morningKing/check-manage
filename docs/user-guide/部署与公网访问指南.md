# 部署指南

本文档说明如何在本地部署 check-manage 项目。

---

## 一、环境要求

| 依赖 | 版本要求 |
|------|----------|
| Node.js | >= 18 |
| Python | >= 3.9 |
| PostgreSQL | >= 14 |

## 二、安装依赖

```bash
# 前端依赖
npm install

# 后端依赖
pip install flask flask-cors psycopg2-binary PyJWT openpyxl werkzeug
```

## 三、初始化数据库

编辑 `server/config.py`，确认数据库连接信息：

```python
DB_CONFIG = {
    'host': 'localhost',
    'dbname': 'casemanage',
    'user': 'postgres',
    'password': 'your-password',
    'port': 5432,
}
```

执行初始化：

```bash
cd server && python init_db.py
```

## 四、启动服务

```bash
# 一键构建 + 启动（推荐）
./start.sh

# 跳过前端构建（已有 dist/ 时）
./start.sh --skip-build
```

启动后本地访问地址：`http://localhost:8080`

默认管理员账号：`admin` / `admin123`

## 五、服务架构

```
浏览器 → localhost:8080 (反向代理)
            ├── 静态文件 ← dist/
            └── /api/*   → Flask 后端 (localhost:3001)
                              └── PostgreSQL (localhost:5432)
```
