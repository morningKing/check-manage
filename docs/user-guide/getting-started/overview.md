# 使用说明

本文档提供系统功能的快速入门指南。复杂功能的详细说明请参阅各专项文档。

---

## 1. 登录与首页

### 登录

访问系统后自动跳转到登录页面。

- **默认管理员账号**：admin / admin123
- 输入用户名和密码后点击「登录」或按 Enter 键

### 首页

首页展示系统概览：
- **统计卡片**：菜单数量、页面配置数、字段配置总数
- **快捷入口**：菜单管理、页面配置的快速跳转链接

## 2. 数据页面

通过左侧菜单导航到各业务数据页面。

### 2.1 视图切换

| 视图 | 说明 |
|------|------|
| **表格视图** | 传统表格，支持排序、筛选、分页 |
| **Excel 视图** | 电子表格，支持冻结、筛选、导出 |
| **看板视图** | 拖拽式卡片布局，按状态分组 |
| **日历视图** | 按日期字段展示事件卡片 |
| **甘特图** | 任务时间轴和依赖关系可视化 |

> **详细说明**：[项目管理功能使用说明](../data/views.md)（看板、日历、甘特图）

### 2.2 基本操作

| 操作 | 说明 |
|------|------|
| **新增** | 点击右上角「+ 新增」，填写表单后保存 |
| **编辑** | 点击「编辑」修改字段值后保存 |
| **删除** | 点击「删除」确认后执行，支持批量删除 |
| **查看详情** | 点击「查看」打开详情弹窗 |
| **导出 Excel** | 点击「导出」下载 .xlsx 文件 |
| **导入 Excel** | 点击「导入」→ 下载模板 → 填写数据 → 上传文件 |

### 2.3 字段类型展示

| 类型 | 表格/详情展示 |
|------|---------------|
| 文本/数值 | 直接显示值 |
| 多行文本 | 保留换行格式 |
| 选项类 | 显示选项标签 |
| 多选类 | Tag 标签展示 |
| 关联/引用 | Tag 展示，点击跳转 |
| 文件 | 文件名列表，可下载 |
| 图片 | 缩略图，支持预览 |

> **数据关系详细说明**：[数据关系类型使用说明](../data/relations.md)

### 2.4 版本管理（管理员）

点击「版本管理」按钮：
- **创建快照**：保存当前数据的点-in-time 副本
- **创建分支**：在独立分支上进行数据修改
- **切换分支**：点击分支名称切换数据视图
- **合并版本**：将分支数据合并回主分支

## 4. 菜单管理

> 系统配置 → 平台管理 → 菜单管理

管理侧边栏菜单结构，支持最多 3 级嵌套。

### 基本操作

- **新增顶级菜单**：创建根菜单
- **添加子菜单**：悬停在节点上点击「添加子菜单」
- **编辑菜单**：选中菜单后在右侧编辑

### 菜单字段

| 字段 | 说明 |
|------|------|
| 菜单名称 | 侧边栏显示名称 |
| 菜单图标 | Element Plus 图标 |
| 路由路径 | 如 /inspection/case |
| 关联页面 | 叶子菜单需选择页面配置 |
| 可见角色 | admin / developer / guest |

> 「系统配置」及其子菜单为内置菜单，不可修改。

## 5. 页面配置

> 系统配置 → 平台管理 → 页面配置

定义数据页面的字段结构和视图配置。

### 5.1 字段配置

| 配置项 | 说明 |
|--------|------|
| 显示名称 | 表单标签和表格列标题 |
| 字段名 | 存储字段名（英文） |
| 控件类型 | 决定输入和展示方式 |
| 必填 | 是否必须填写 |
| 主键 | 唯一标识字段 |
| 隐藏 | 在表格中隐藏 |
| 默认值 | 新增记录时预填的值。单选/多选下拉、单选/复选框可从已配置选项中选择；文本/数字/日期等按类型直接填写；留空表示无默认值 |

### 5.2 支持的控件类型

| 类型 | 说明 | 特殊配置 |
|------|------|---------|
| text | 单行文本 | - |
| textarea | 多行文本 | - |
| richText | 富文本编辑器 | - |
| markdown | Markdown 编辑器（实时预览，查看时前端渲染） | - |
| number | 数值输入 | - |
| select | 单选下拉 | 选项来源配置 |
| multiSelect | 多选下拉 | 选项来源配置 |
| date | 日期选择 | - |
| datetime | 日期时间 | - |
| file | 文件上传 | 允许的文件类型 |
| image | 图片上传 | 允许的文件类型 |
| **relation** | 多对多关联 | 目标集合、反向字段 |
| **reference** | 一对多引用 | 继承字段配置 |
| **quoteSelect** | 引用选择 | 目标集合配置 |
| autoTimestamp | 自动时间戳 | 创建/更新时自动填充 |
| autoSequence | 自增序列 | 前缀和上限配置 |

> **数据关系配置详解**：[数据关系类型使用说明](../data/relations.md)

**文件/图片上传的类型约束**：编辑 `file`/`image` 字段时可配置「允许的文件类型」（输入扩展名如 `pdf`、`.docx` 后回车添加，留空表示不限制）。约束在前端（选择文件对话框的可选类型、上传前的即时校验）和后端（`POST /data-files/upload` 按 `collection`+`fieldName` 查字段配置二次校验）两处生效，避免绕过前端页面直接调接口上传任意类型文件。字段的「占位提示」配置也会显示为上传区域下方的提示文案（留空则显示按当前类型约束自动生成的默认提示）。

### 5.3 视图配置

在「视图配置」区域可配置：

| 视图 | 必需配置 |
|------|---------|
| 看板 | 分组字段、卡片标题、颜色字段 |
| 日历 | 日期字段、卡片标题、颜色字段 |
| 甘特图 | 开始日期、结束日期、标题字段 |

> **视图配置详解**：[项目管理功能使用说明](../data/views.md)

## 6. 用户管理

| 角色 | 权限范围 |
|------|---------|
| admin | 所有功能，包括系统配置 |

## 8. 导出脚本

> 系统配置 → 数据工具 → 导出脚本

自定义数据导出格式和内容。

### 配置项

| 字段 | 说明 |
|------|------|
| 脚本名称 | 显示在导出菜单中 |
| 输出格式 | json / csv / xml / txt / html |
| 导出维度 | 整页数据 或 单行数据 |

### 脚本变量

| 变量 | 类型 | 说明 |
|------|------|------|
| data | list[dict] | 业务数据列表 |
| fields | list[dict] | 字段配置列表 |
| result | str/bytes | **必须** — 导出内容 |
| filename | str | 可选 — 文件名 |

## 11. Open API

> 系统配置 → 数据工具 → Open API

为外部系统提供数据访问接口。

### 创建 API Key

1. 点击「创建 API Key」
2. 输入密钥名称
3. **密钥仅显示一次，请立即保存**

### 调用示例

```
GET /api/v1/collections
  → 返回已开启 Open API 的集合列表

GET /api/v1/collections/{collection}
  → 返回指定集合的数据

请求头：
  X-API-Key: cm_xxxxxxxxxx
```

> **OpenAPI 详细文档**：[OpenAPI接口文档](../integration/open-api.md)

## 19. 常见操作流程

### 创建新业务页面

1. **页面配置** → 新增 → 定义字段 → 保存
2. **菜单管理** → 添加子菜单 → 关联页面 → 设置角色
3. 刷新页面，新菜单出现

### 建立数据关联

1. **页面配置** → 选择源页面 → 添加 relation 字段
2. 配置目标集合、显示字段、反向字段名
3. 在另一侧页面添加对应的反向关联字段

> 详细步骤见：[数据关系类型使用说明](../data/relations.md)

### 配置看板/日历/甘特图

1. **页面配置** → 选择页面 → 视图配置区域
2. 选择对应标签页（看板/日历/甘特图）
3. 配置必需字段（分组/日期/标题等）
4. 设置默认视图
5. 数据页切换查看

> 详细配置见：[项目管理功能使用说明](../data/views.md)

## 注意事项

1. **密码安全**：首次部署后立即修改默认管理员密码
2. **备份策略**：建议启用每日定时备份，保留至少 7 天
3. **字段名规范**：只能使用英文字母、数字和下划线，以字母开头
4. **角色分配**：遵循最小权限原则
5. **浏览器支持**：推荐 Chrome、Edge 等现代浏览器
6. **系统菜单**：「系统配置」菜单为内置菜单，不可修改删除
7. **脚本安全**：脚本在沙箱中执行，超时自动终止
8. **API Key**：创建后仅显示一次，不用的密钥及时禁用
9. **并发编辑**：系统通过乐观锁保护数据一致性
10. **自增序列**：删除中间记录不会回收编号
