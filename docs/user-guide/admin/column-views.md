# 列视图（保存的列配置）

在数据页的**表格视图**中，你可以自定义列的显隐、顺序、宽度与筛选条件，并将其保存为「列视图」，下次打开自动恢复。列视图按页面独立保存（互不影响）。

## 配置列

1. 打开任意数据页的 **表格视图**。
2. 点击工具栏的 **列设置**（齿轮 / 列图标）。
3. 在弹出面板中：
   - **勾选 / 取消** 列以控制显隐；
   - **拖拽** 调整列顺序；
   - 拖动列边界调整 **列宽**；
   - 设置某些列的 **筛选条件**。

## 保存与切换列视图

- 调整后点击 **保存视图**，命名后即生成一个列视图。
- 同一页面可保存多个列视图，在工具栏下拉中快速切换。
- 列视图配置保存在后端（`column_views` 表），换浏览器 / 设备也能恢复。

## 注意事项

- 列视图**按页面隔离**：切换数据页时会重新加载该页自己的列视图，不会串用其它页的配置。
- 若新增 / 删除了字段，已保存的列视图会自动忽略不存在的列。
