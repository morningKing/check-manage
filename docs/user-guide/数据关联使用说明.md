# 数据关联使用说明

## 概述

数据关联功能用于在两个动态数据页之间建立**多对多双向关联**关系。

例如：「巡检用例」和「巡检模板」之间互相关联——一个用例可以属于多个模板，一个模板也可以包含多个用例。当在一侧修改关联时，另一侧自动同步。

---

## 前置准备

使用关联功能前，需要先在**系统配置 > 页面配置**中完成以下准备：

1. 创建两个数据页的页面配置（如「巡检用例」和「巡检模板」）
2. 每个页面至少有一个普通字段用于显示（如「用例名称」「模板名称」）
3. 在**系统配置 > 菜单管理**中为两个页面分别创建菜单入口

---

## 配置步骤

以「巡检用例」与「巡检模板」互相关联为例。

### 第一步：给巡检模板添加关联字段

1. 进入 **系统配置 > 页面配置**
2. 在左侧列表选中「巡检模板」
3. 在右侧字段配置区域点击 **添加字段**
4. 填写字段信息：
   - **显示名称**：`关联用例`
   - **字段名**：`cases`（英文，用于数据存储）
   - **控件类型**：选择 `关联选择`
5. 底部出现**关联配置**区域，填写：
   - **目标集合**：下拉选择 `巡检用例`
   - **显示字段**：下拉选择 `用例名称（caseName）`
   - **反向字段**：输入 `templates`（稍后在巡检用例侧创建的字段名）
6. 点击 **确定** 保存字段
7. 点击 **保存配置** 将字段持久化到服务器

### 第二步：给巡检用例添加关联字段

1. 在左侧列表选中「巡检用例」
2. 点击 **添加字段**，填写：
   - **显示名称**：`所属模板`
   - **字段名**：`templates`（与上一步的反向字段名一致）
   - **控件类型**：选择 `关联选择`
3. 关联配置：
   - **目标集合**：下拉选择 `巡检模板`
   - **显示字段**：下拉选择 `模板名称（templateName）`
   - **反向字段**：下拉选择 `关联用例（cases）`（已存在的关联字段会出现在下拉列表中）
4. 点击 **确定**，再点击 **保存配置**

> **关键点**：两侧的字段名和反向字段名必须互相对应。即 A 侧的字段名 = B 侧的反向字段名，反之亦然。

### 配置对照表

| 配置项 | 巡检用例侧 | 巡检模板侧 |
|--------|-----------|-----------|
| 显示名称 | 所属模板 | 关联用例 |
| 字段名 | `templates` | `cases` |
| 目标集合 | 巡检模板 | 巡检用例 |
| 显示字段 | 模板名称 | 用例名称 |
| 反向字段 | `cases` | `templates` |

---

## 使用方式

### 新增记录时选择关联

1. 进入任一数据页（如「巡检用例」），点击 **新增**
2. 填写普通字段后，在「所属模板」关联字段中勾选要关联的模板记录
3. 点击 **确定** 提交

此时：
- 巡检用例记录保存成功，其「所属模板」字段显示所选模板数量
- 切换到「巡检模板」页面，对应模板记录的「关联用例」字段**自动包含**刚创建的用例

### 编辑记录时修改关联

1. 点击某条记录的 **编辑** 按钮
2. 修改关联字段的选中项（增加或移除勾选）
3. 点击 **确定** 提交

系统会自动处理双向同步：
- 新增的关联：对方记录自动添加反向关联
- 移除的关联：对方记录自动移除反向关联

### 删除记录时清理关联

删除一条记录后，该记录在所有关联方的引用会被自动清除。

---

## 双向同步原理

以修改「用例A」的所属模板从 `[模板1, 模板3]` 改为 `[模板1, 模板2]` 为例：

```
操作前：
  用例A.templates = [模板1, 模板3]
  模板1.cases = [用例A]
  模板2.cases = []
  模板3.cases = [用例A]

用户操作：将用例A的所属模板改为 [模板1, 模板2]

系统自动处理：
  1. 用例A.templates → [模板1, 模板2]     ← 直接更新
  2. 模板1.cases → [用例A]                 ← 不变（未移除）
  3. 模板2.cases → [用例A]                 ← 自动添加（新增关联）
  4. 模板3.cases → []                      ← 自动移除（取消关联）

操作后：
  用例A.templates = [模板1, 模板2]
  模板1.cases = [用例A]
  模板2.cases = [用例A]
  模板3.cases = []
```

---

## 表格中的显示

关联字段在数据表格中以 `N 条关联` 的形式显示（如 `3 条关联`），无关联时显示 `-`。

---

## 注意事项

1. **字段名对应**：两侧配置的字段名和反向字段名必须严格匹配，否则双向同步不生效
2. **先创建页面配置**：目标集合的页面配置必须已存在，才能在下拉中选择
3. **先有数据再关联**：关联选择控件会加载目标集合的现有记录作为选项，需要目标集合中已有数据
4. **关联数据独立存储**：关联关系存储在独立的关联表中，不影响记录本身的数据字段
